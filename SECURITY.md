# Security policy

## Reporting a vulnerability

Please report suspected security vulnerabilities privately to
`assaf@californiacarecompass.com`. Include reproduction steps and the affected
version or commit when possible. Please do not include sensitive details in a
public GitHub issue.

Non-sensitive correctness and data-quality reports may be filed through the
repository's public issue tracker.

## Scope

This server is read-only, bundles public data, accepts no credentials, and
makes no network requests at runtime. Reports concerning dependency behavior,
tool-input handling, container execution, or inaccurate security claims are in
scope.
